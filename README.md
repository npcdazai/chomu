# Valentine Next.js Project

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- App Router is used (`src/app`).
- Main content is in `src/app/page.tsx`.
- Styling is in `src/app/globals.css`.
- Replace `public/photo-placeholder.svg` with your own image (same file name), or update the `img` path in `src/app/page.tsx`.
